Scratch Project

The Legend of Thanos

Computer Science Principles

September 2018

The objective of this project was to create a themed platformer game with death and respawning across multiple levels. I developed and coded this game with Nibodh Vallapuredy in Scratch. It was necessary for us to research metroid-vania games. The greatest challenge was combining our ideas after working separately. Since both of us were unfamiliar with the programming language, it was necessary to explain every line of code to each other. By the end of the project we had become proficient in Scratch and learned how to collaborate more efficiently.

Instructions:

    A and D to move left and right
    W to jump
    Collect the Infinity Stones!

Image Manipulation Project

Spicy Portraits Inc.

Computer Science Principles

March 2019

My partner for this project was Megan Fannin. We used cloud nine to code this in the python language. This program is able to take any image and add a customizable border, symbol, and geometric pattern to it. The most difficult part of making it was figuring out how to open and save files outside the working directory. Through this project we both gained a better understanding of functions and code from the python library.
